# hiking-scraper

This is using office libraries, so it is tested only on Windows.
Python3 is used.

## venv setup

```
python -m venv venv
venv\scripts\activate.bat
```

## install dependencies

```
pip install -r requirements.txt
```

## run scraper

```
python planine.py
```

## others  

pip upgrade  
```
python -m pip install --upgrade pip
```
